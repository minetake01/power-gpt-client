# PowerGPTClient
プラグインシステムを搭載した、OpenAI GPT APIのクライアントGUIアプリです。

## Building
1. リポジトリのクローン<br>
`git clone https://github.com/minetake01/power-gpt-client.git && cd power-gpt-client`
2. 依存関係のインストール<br>
`pnpm install`
3. ビルド・実行<br>
    - 開発ビルド<br>
    `pnpm tauri dev`
    - リリースビルド<br>
    `pnpm tauri build`
